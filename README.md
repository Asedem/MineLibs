# MineLibs

<hr>

## Installation

#### Maven
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependency>
    <groupId>com.github.Asedem</groupId>
    <artifactId>MineLibs</artifactId>
    <version>fc5dda2dae</version>
</dependency>
```

#### Gradle
```gradle
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.Asedem:MineLibs:fc5dda2dae'
}
```
