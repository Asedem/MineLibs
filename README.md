# MineLibs

<hr>

## Installation

#### Repository
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

#### Dependency
```xml
<dependency>
    <groupId>com.github.Asedem</groupId>
    <artifactId>MineLibs</artifactId>
    <version>Tag</version>
</dependency>
```
